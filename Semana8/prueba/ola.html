Programación Web Avanzada

Curso 2024 - Primer Semestre

Descripción del Curso

Este curso está diseñado para proporcionar a los estudiantes una comprensión profunda de las tecnologías web modernas y las mejores prácticas en el desarrollo web. A lo largo del semestre, exploraremos conceptos avanzados de programación front-end y back-end.

Objetivos de Aprendizaje

  • Dominar HTML5, CSS3 y JavaScript moderno
  • Desarrollar aplicaciones web responsivas y accesibles
  • Implementar bases de datos y APIs RESTful
  • Comprender frameworks modernos de desarrollo web
  • Aplicar principios de seguridad web
